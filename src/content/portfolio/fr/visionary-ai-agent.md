## Présentation du projet

Agent conversationnel intelligent spécialisé dans l'assistance technique pour API d'analyse d'images Visionary.ai.

## Défi

Transformer l'expérience documentation statique en assistant conversationnel qui comprend le contexte technique et répond précisément aux questions développeurs.

## Solution

Système hybrid conversational combinant:

- Knowledge base structurée par domaines techniques
- Génération IA contextuelle via Gemini
- Interface chat moderne et responsive
- Fallbacks intelligents pour garantir la fiabilité

## Livrables clés

- Interface conversationnelle complète (React + TypeScript)
- Intégration API Gemini avec gestion fallbacks
- Architecture serverless (Netlify Functions)
- Documentation technique du système
- Mode dark/light, historique conversationnel

## Résultats

Proof of concept fonctionnel démontrant l'approche "from documentation to conversation". Réponses instantanées (< 30 secondes) et contextuelles adaptées au parcours utilisateur.

## Outils

- React 18 + TypeScript
- API Gemini
- Netlify Functions
- Tailwind CSS
- CI/CD automatisé
